# Modern Todo App

A modern and responsive Todo application built with HTML, Tailwind CSS, and Vanilla JavaScript.

## Features

* Add, edit, and delete tasks
* Mark tasks as completed
* Task categories
* Priority levels
* Due dates
* Search functionality
* Filter tasks
* Statistics dashboard
* Drag-and-drop sorting
* Toast notifications
* Dark mode persistence
* Local storage support
* Glassmorphism UI
* Responsive design

## Technologies Used

* HTML5
* Tailwind CSS
* Vanilla JavaScript (ES6 Modules)
* Local Storage API
* SortableJS
* ToastifyJS
* Font Awesome
